# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-01-XX

### Added
- Initial release of AI Threat Intelligence Dashboard
- Real-time global attack map visualization
- Live threat feed with filtering and search
- AI-based threat severity scoring
- IP reputation checker
- URL and file hash malware scanner
- MITRE ATT&CK tactic mapping
- Alert prioritization using machine learning
- Interactive charts for attack trends, threat types, and affected industries
- Login and role-based access control (Admin, Analyst, Viewer)
- Main dashboard with key metrics
- Incident response panel
- Threat hunting workspace
- Reports & analytics with PDF export
- Settings & API management
- Dark cybersecurity theme with neon blue and red accents
- Responsive design for mobile and desktop
- Real-time data refresh functionality
- Exportable PDF reports

### Technical
- React 18 with TypeScript
- Vite build tool
- Tailwind CSS for styling
- Recharts for data visualization
- Leaflet for map visualization
- React Router for navigation
- jsPDF for PDF generation
